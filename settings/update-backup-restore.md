# Update, Backup & Restore

On this page you have the option to update your board using over the air using a firmware file found under releases, download a backup file of your settings and also to restore these settings.
<br><br>

## Firmware Update
The firmware files can be found [here](https://github.com/derDeno/PandaLED/releases). You can update to a newer version or downgrade to an older version.
If an update is available, you will be informed on the home page.
